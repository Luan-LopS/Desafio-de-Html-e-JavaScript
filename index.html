Desafio de validação de compo com javaScript

1 Crie um formulário HTML que possua dois campos numéricos: campo A campo B 2 Insira um botão para submeter o formulário. 3 Crie uma validação no JavaScript, na qual o formulário será válido caso o número B seja maior que o número A. 4 Exiba uma mensagem positiva para o usuário quando o formulário for válido e uma mensagem negativa quando for inválido. 5 Suba no Github no repositório do curso em uma branch chamada exercicio_html_js. 6 Copie o link do repositório na plataforma da EBAC que a equipe de tutoria dará o feedback.
